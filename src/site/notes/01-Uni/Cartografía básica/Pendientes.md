# Pendientes
- Se miden en grados.
- Se pueden usar para varias cosas como tablas de población o de uso de la tierra.
- Las pendientes hacen que un camino sea más o menos largo según su grado.
Distancia reducida: Distancia de un punto a otro.
Desnivel: Elevación de un punto a otro.
Distancia geométrica: raíz cuadrada de cateto^2 + cateto^2.
La distancia real es la geométrica pero con variaciones.
- Las pendientes se expresan en **grados sexagesimales** o como **porcentaje**.
